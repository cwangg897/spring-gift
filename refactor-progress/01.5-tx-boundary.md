## [선택 2] 트랜잭션 경계도

```
Before: Controller → repo.save x3 + kakao (NO TX)
After:  Controller → OrderFacade.createOrder() ──┐
                                                  ├─ [ @Transactional ]
                                                  │     repo.save x3 (atomic)
                                                  └─ kakao (현재는 동기, 06-B에서 이벤트화)
```
